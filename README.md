## Introduction

## Building

Build default target
    
    make all
    
Build Verilator target
    
    make ver-build
    
Build Icarus target
    
    make iver-build
    
Build Modelsim target
    
    make msim-build

## Run test

Run default test example
    
    make test

Run Modelsim test example

    make msim-test
